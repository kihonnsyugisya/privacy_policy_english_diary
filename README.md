# 英語日記 - プライバシーポリシー・利用規約サイト

「英語日記」アプリのプライバシーポリシーと利用規約を掲載するWebサイトです。

## 🌐 公開URL

https://kihonnsyugisya.github.io/privacy_policy_english_diary/

## 📁 ファイル構成

```
.
├── index.html          # ランディングページ
├── privacy.html        # プライバシーポリシー
├── terms.html          # 利用規約
├── css/
│   ├── common.css      # 共通スタイル（CSS変数、基本リセット）
│   ├── index.css       # ランディングページ専用スタイル
│   └── policy.css      # プライバシーポリシー・利用規約ページ用スタイル
└── README.md           # このファイル
```

## 📄 ページ構成

### ランディングページ（index.html）
- アプリの紹介
- 主な機能の説明
- プレミアムプランの紹介
- 利用規約・プライバシーポリシーへのリンク

### プライバシーポリシー（privacy.html）
- 適用範囲
- 収集する情報
- 情報の利用目的
- データの保存・管理
- 広告について
- サブスクリプション
- 外部サービスの利用
- その他関連情報

### 利用規約（terms.html）
- 本規約について
- サービスの利用開始
- データの保存と管理
- 広告について
- 決済およびサブスクリプション
- 権利帰属および利用許諾
- 禁止行為
- その他関連条項

## 🛠️ 技術スタック

- HTML5
- CSS3（CSS変数、Flexbox、Grid）
- レスポンシブデザイン対応

## 📱 レスポンシブ対応

すべてのページはモバイルデバイスに対応しています。

- デスクトップ：最大幅900px
- タブレット・モバイル：画面幅に応じて自動調整

## 🎨 デザイン

- カラーパレット：温かみのあるクリーム色の背景（#fdf8e7）をベースに、アクセントカラーとしてブルー（#3498db）を使用
- フォント：システムフォント（San Francisco、Segoe UI、Hiragino Kaku Gothic ProN など）
- アニメーション：ホバーエフェクト、トランジション
